# File Analyzer in Python

This program analyzes a text file and displays:
- number of lines
- number of words
- number of characters
- most frequent word
- top 3 most frequent words
- word search

## Features
- text analysis
- word frequency
- search functionality

## Concepts used
- loops (while)
- conditions (if)
- lists
- file handling

## Execution
python main.py
